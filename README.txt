Welcome to IDA Pomidor, a productivity plugin for Hex-Ray's IDA Pro disassembler.

To install IDA Pomidor simply copy 'idapomidor.py' and 'idapomidor' folder to IDA's plugins folder. The plugin will be automatically loaded the next time you start IDA Pro.

You can find the latest IDA Pomidor version and documentation here: 
    http://thesprawl.org/projects/ida-pomidor/
